# Quiz app
A quiz app built with flask, jinja and sqlite3


## Steps to run app
1. Create virtual environment with 
`bash
    python -m venv env
`

2. Activate virtual env
`bash
    source env/bin/activate
`

3. Install dependencies with pip

4. Create db
`bash
    python create_db.py
`

5. Run application 
`bash
    python quizzy.py
`
